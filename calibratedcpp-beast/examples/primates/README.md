# Primates Analyses

Phylogenetic analysis of 29 primates (+ outgroups) using the Calibrated Coalescent Point Process.
Calibration bounds are defined in [`primates_uniform_bounds.csv`](primates_uniform_bounds.csv).

## Overview

The workflow produces 8 XML files from two base analyses:

| | Conditioned | Not-conditioned |
|---|---|---|
| **Consistent prior + sequences** | `primates_consistent_prior.xml` | `primates_consistent_prior_not-conditioned.xml` |
| **Consistent prior, sample from prior** | `sample-from-prior_primates_consistent_prior.xml` | `sample-from-prior_primates_consistent_prior_not-conditioned.xml` |
| **Uniform MRCA prior + sequences** | `primates_uniform_prior.xml` | `primates_uniform_prior_not-conditioned.xml` |
| **Uniform MRCA prior, sample from prior** | `sample-from-prior_primates_uniform_prior.xml` | `sample-from-prior_primates_uniform_prior_not-conditioned.xml` |

- **Consistent prior**: calibrations encoded as `ConditionedMRCAPrior` (from `primates.lphy`)
- **Uniform MRCA prior**: calibrations replaced with independent `Uniform` `MRCAPrior` distributions
- **Not-conditioned**: `conditionOnCalibrations="false"` on `CalibratedBirthDeathSkylineModel`
- **Sample from prior**: sequence data replaced with `"?"` so the likelihood is not evaluated

## Step-by-step workflow

The XMLs were generated by the following procedure.

### 1. Generate the consistent prior XML

Run lphybeast from the repo root to convert `primates.lphy` into a BEAST2 XML:

```bash
mvn -pl calibratedcpp-lphybeast exec:exec \
  -Dlphybeast.args="convert -o primates_consistent_prior.xml \
    ../calibratedcpp-beast/examples/primates/primates.lphy"
```

This produces `primates_consistent_prior.xml` with the `ConditionedMRCAPrior` calibration prior.

### 2. Generate the uniform MRCA prior XML

Convert the consistent prior XML to a uniform MRCA prior XML. This replaces the
`CalibrationPrior` distribution with individual `MRCAPrior(Uniform)` distributions and
adds log entries for each MRCA prior to the file logger:

```bash
python3 calibratedcpp-beast/examples/primates/change_to_uniform_prior.py
```

Output: `primates_uniform_prior.xml`

### 3. Sample a starting tree for the uniform MRCA prior analysis

The uniform MRCA prior uses hard bounds, so the initial tree must satisfy them.
Sample a tree from the narrow calibration prior using SLPhy:

```bash
mvn -pl calibratedcpp-lphy-studio exec:exec \
  -Dlphy.args="-r 1 ../calibratedcpp-beast/examples/primates/primates_narrow.lphy"
```

This writes `primates_narrow_tree.trees` (NEXUS format) to the primates directory.

### 4. Replace the initial tree in the uniform MRCA prior XML

```bash
python3 calibratedcpp-beast/examples/primates/change_initial_tree.py
```

This reads the first tree from `primates_narrow_tree.trees` and overwrites the
`stateNode[@id="tree"]` newick in `primates_uniform_prior.xml`.

### 5. Generate all 8 XML variants

```bash
python3 calibratedcpp-beast/examples/primates/make_sample_from_prior.py
```

Produces all 8 files from the two base XMLs by applying every combination of
conditioned/not-conditioned and with-sequences/sample-from-prior.

## Scripts

| Script | Input | Output |
|---|---|---|
| `change_to_uniform_prior.py` | `primates_consistent_prior.xml` | `primates_uniform_prior.xml` |
| `change_initial_tree.py` | `primates_narrow_tree.trees` → `primates_uniform_prior.xml` | `primates_uniform_prior.xml` (modified in place) |
| `make_sample_from_prior.py` | `primates_consistent_prior.xml`, `primates_uniform_prior.xml` | 8 XML files |
